Про програму

Payment Calculator — це інструмент для розрахунку здачі при оплаті готівковою валютою (USD або EUR), якщо сума чеку виражена у гривнях. Програма автоматично завантажує офіційний курс НБУ через публічне API та курс готівкового обмінника через Telegram-канал, і одразу підставляє актуальний курс як робочий. Касир вводить суму чеку та суму, яку передав клієнт у валюті, — і миттєво бачить скільки здачі повернути або яка залишається недоплата. Курс зберігається між сесіями, тому програма залишається корисною навіть без підключення до інтернету.

Блоки інтерфейсу
💱
Курси валют
Відображає курс НБУ (офіційний) та курс обмінника (готівковий) для USD та EUR. Робочий курс можна відредагувати вручну. Кнопки «Застосувати курс НБУ» та «Застосувати обмінник» одним кліком підставляють потрібне значення.
🪙
Валюта оплати
Перемикач між USD і EUR — великі кнопки для швидкого вибору. Зміна валюти миттєво перераховує всі значення та оновлює підписи у блоці підсумку.
🧾
Сума чеку
Поле введення суми у гривнях. Після кожного введеного символу автоматично розраховується еквівалент у вибраній валюті за робочим курсом.
💵
Оплата клієнта
Поле для введення суми, яку передав клієнт у валюті. Дані з цього поля використовуються для розрахунку здачі або недоплати.
📊
Підсумок
Показує суму до сплати, отриману від клієнта суму та здачу у валюті і у гривневому еквіваленті. При недоплаті заголовок і значення підсвічуються червоним кольором із позначкою ⚠.
Особливості
  • Автоматичне отримання курсу НБУ при запуску через офіційне API bank.gov.ua
  • Парсинг курсу готівкового обмінника з публічного Telegram-каналу — без реєстрації та токенів
  • Пріоритет курсу обмінника як робочого: готівковий курс зазвичай відрізняється від офіційного НБУ
  • Ручне коригування робочого курсу — поле редагується напряму, зміна миттєво перераховує підсумок
  • Швидкі кнопки підстановки «Застосувати курс НБУ» та «Застосувати обмінник» для переключення між джерелами
  • Перерахунок у реальному часі: результат оновлюється після кожного введеного символу без натискання кнопок
  • Збереження курсів і останнього стану у файл settings.json поруч із програмою — працює без інтернету із збереженим курсом
  • Захист від некоректного введення: фільтрація символів безпосередньо при парсингу значень полів
  • Індикація часу останнього оновлення курсів; при відсутності з'єднання — попередження жовтим кольором
  • Кнопка «Скинути все» очищає поля чеку та оплати без скидання курсів
