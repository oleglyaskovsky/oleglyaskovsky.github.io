Про програму

FFmpeg Manager — це зручний графічний інтерфейс для інтеграції FFmpeg у контекстне меню Windows. Замість того щоб кожного разу відкривати термінал і вручну вводити команди, достатньо клацнути правою кнопкою миші на відео-, аудіофайлі або зображенні — і потрібна операція з'явиться прямо в меню Провідника. Програма записує обрані команди до реєстру Windows (HKCU) і автоматично прив'язує їх до відповідних типів файлів, після чого FFmpeg стає доступним у два кліки для будь-якого медіафайлу в системі.

Категорії команд
🎬 Відео
Стиснення (H.264, H.265), обрізання, поворот, дзеркалення, зміна роздільної здатності (480p–4K), зміна FPS, прискорення/сповільнення, реверс, стабілізація, видалення метаданих
🎵 Аудіо
Витяг аудіодоріжки (MP3, WAV, FLAC, AAC), нормалізація гучності, зміна бітрейту, частоти дискретизації, конвертація між моно/стерео, видалення тиші, шумозаглушення, ефекти fade in/out
🔄 Конвертація
Конвертація між форматами: MP4, MKV, AVI, WebM, MOV, TS, MP3, WAV, FLAC, OGG, AAC, Opus, GIF, а також витяг першого кадру у PNG або JPG
🖼️ Зображення та кадри
Створення GIF (звичайне та HQ), витяг кадрів (по 1/сек або всіх), мініатюра з відео, додавання водяного знаку, мозаїка прев'ю 4×4
🎨 Фільтри
Чорно-біле, сепія, вінтаж, негатив, яскравість, контраст, насиченість, розмиття, різкість, шумозаглушення, деінтерлейсинг, теплі/холодні відтінки, вінь'єтка, виявлення країв
📝 Субтитри
Вшивання субтитрів у відео (hardcoded), додавання субтитрів як окремої доріжки (soft), витяг вбудованих субтитрів у файл SRT
ℹ️ Інформація
Детальна інформація про файл через ffprobe: кодеки, роздільна здатність, тривалість, бітрейт, метадані (теги)
🌐 Стрімінг
Розбиття відео на сегменти HLS для потокового відтворення, створення DASH маніфесту для адаптивного стрімінгу
Особливості
  • Інтеграція безпосередньо у Провідник Windows — команди FFmpeg доступні правою кнопкою миші на будь-якому медіафайлі
  • Більше 70 готових FFmpeg-команд, організованих у 8 категорій: відео, аудіо, конвертація, зображення, фільтри, субтитри, інформація, стрімінг
  • Індикатор наявності FFmpeg — програма перевіряє встановлення при запуску та пропонує посилання на завантаження якщо FFmpeg не знайдено
  • Виконання команд у фоновому режимі з окремим вікном прогресу, живим виводом FFmpeg та кнопкою скасування
  • Команди з позначкою «📎 +файл» відкривають діалог вибору другого файлу (субтитри, аудіодоріжка, водяний знак)
  • Команди з позначкою «⌨️ параметри» відкривають форму введення (наприклад, час початку та тривалість для довільного обрізання)
  • Інформаційні команди (ℹ️) виводять результат роботи ffprobe у зручному текстовому вікні без збереження файлу
  • Кнопки «Вибрати все» та «Зняти все» для зручного керування в межах кожної категорії
  • Записи реєстру вносяться лише до HKCU — без змін у системних гілках, не вимагаються права адміністратора
  • Кнопка «Оновити Провідник» перезапускає explorer.exe для миттєвого застосування змін у контекстному меню
  • Підтримка ffmpeg.exe поруч із програмою як альтернатива системному PATH — корисно для портативного використання
