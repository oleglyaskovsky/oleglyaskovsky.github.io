Про програму

Font Restorer & Cleaner — це утиліта для наведення порядку зі шрифтами у Windows після встановлення стороннього програмного забезпечення, яке нерідко залишає у системі десятки зайвих шрифтів. Програма порівнює вміст системної папки шрифтів і папки користувача з вбудованим білим списком стандартних шрифтів Windows 10/11, виявляє сторонні файли, створює їх резервну копію та видаляє разом із записами у реєстрі. Окремий інструмент відновлення повертає стандартні значення FontSubstitutes, вмикає ClearType та перезапускає кеш шрифтів — все це без зайвих діалогів і ручного редагування реєстру.

Основні функції
🔍
Аналіз
Сканує системну папку C:\Windows\Fonts та папку шрифтів поточного користувача. Порівнює знайдені файли з білим списком стандартних шрифтів Windows і виводить перелік сторонніх шрифтів з кількістю у кожному джерелі.
🧹
Експорт та Очищення
Копіює всі сторонні шрифти у папку Font_Backup поруч із програмою, після чого видаляє файли та відповідні записи з реєстру Windows (HKLM та HKCU). Якщо файл зайнятий — повідомляє про необхідність перезавантаження.
🔧
Відновити реєстр
Відновлює стандартні записи FontSubstitutes (Arial, Tahoma, Times New Roman та ін.), вмикає ClearType та згладжування шрифтів, зупиняє службу FontCache, видаляє файл кешу FNTCACHE.DAT та перезапускає службу.
Особливості
  • Вбудований білий список із понад 300 стандартних файлів шрифтів Windows 10/11: TrueType (.ttf/.ttc), OpenType (.otf) та растрові шрифти (.fon) для DOS/CMD
  • Сканування двох джерел одночасно — системної папки шрифтів і папки шрифтів поточного користувача (ручне встановлення через HKCU)
  • Автоматичний бекап перед видаленням — всі сторонні шрифти зберігаються у папку Font_Backup поруч із програмою
  • Очищення реєстру: видаляються записи з гілок HKLM та HKCU відповідно до джерела шрифту
  • Відновлення FontSubstitutes: повертає стандартні замінники шрифтів (MS Shell Dlg → Microsoft Sans Serif, Helvetica → Arial тощо)
  • Скидання ClearType: повертає стандартне згладжування шрифтів через параметри Control Panel\Desktop
  • Очищення кешу шрифтів: зупиняє службу FontCache, видаляє FNTCACHE.DAT та перезапускає службу
  • Виконання операцій у фоновому потоці — інтерфейс не заморожується під час роботи
  • Живий журнал з часовими мітками та рівнями повідомлень (INFO, RESULT, OK, WARN, ERROR)
  • DPI-aware центрування вікна: коректно відображається на екранах з масштабуванням 125%, 150%, 200%
  • Запускається лише з правами адміністратора — програма автоматично запитує підвищення привілеїв через UAC
