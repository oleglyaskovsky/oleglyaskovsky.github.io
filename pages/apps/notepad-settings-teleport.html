Notepad++ Settings Teleport

Перенесення налаштувань Notepad++: Portable → Installed
Завантажити з MEGA
Про програму

Notepad++ Settings Teleport — це невеличкий інструмент для швидкого перенесення налаштувань Notepad++ з portable-версії у встановлену (installed) версію редактора. Замість того щоб вручну шукати й копіювати конфігураційні файли, теми та плагіни, достатньо вказати шлях до portable-папки й натиснути одну кнопку — програма сама запакує все потрібне в один ZIP-архів, а при імпорті розкладе його по правильних папках встановленого Notepad++, автоматично визначаючи, куди саме — у %APPDATA% чи в Program Files — можна писати плагіни.

Що переноситься
Конфігураційні файли
config.xml, contextMenu.xml, shortcuts.xml, langs.xml, stylers.xml
Папки
themes, plugins, backup — з повним вмістом і структурою підпапок
Формат архіву
Один ZIP-файл Npp_Backup_Data.zip, що зберігається поруч із програмою
Куди відновлюється
Конфіги і теми → %APPDATA%\Notepad++, плагіни → Program Files або AppData (залежно від прав запису)
Особливості
  • Кнопка «Експорт» пакує всі налаштування Portable Notepad++ в один ZIP-архів одним кліком
  • Кнопка «Імпорт» автоматично розпаковує архів у встановлену версію редактора
  • Автоматичний пошук папки Portable Notepad++ поруч із програмою (за наявністю notepad++.exe)
  • Перевірка, чи запущений Notepad++, перед імпортом — захищає від пошкодження налаштувань під час перезапису файлів
  • Автовизначення доступного для запису розташування для плагінів: спочатку пробує Program Files, за відсутності прав — переходить на AppData
  • Автоматичний пошук архіву бекапу поруч із програмою при імпорті, а якщо не знайдено — вибір файлу через провідник
  • Підтримка тихого режиму через ключі командного рядка /S або /Silent — для автоматизованого розгортання без GUI, наприклад у власних інсталяторах
  • Детальний журнал операцій у реальному часі з переліком кожного скопійованого файлу чи папки та розміром готового архіву
