Про програму

Software Installer — це зручний інструмент для швидкого розгортання Windows-середовища після чистої установки системи. Замість того щоб шукати кожну програму окремо в інтернеті, достатньо відзначити потрібні галочки і натиснути одну кнопку — програма сама завантажить і встановить усе вибране у фоновому режимі, паралельно та без зайвих кліків. Підтримуються як звичайні інсталятори через WinGet, так і власні архіви з налаштованими конфігураціями.

Особливості
  • Паралельна установка кількох програм одночасно без черги очікування
  • Підтримка WinGet: автоматична перевірка наявності та можливість встановити або оновити прямо з інтерфейсу
  • Установка Microsoft Visual C++ Redistributable одразу у двох розрядностях (x86 + x64) за один клік
  • Завантаження Microsoft Office будь-якої версії через офіційний C2R-канал з автоматичним визначенням розрядності системи
  • Утиліти підтримують два режими: ЛКМ — завантажити і запустити, ПКМ — зберегти portable-версію в папку C:\Portable
  • Блок "Інформація про систему" показує версію Windows, модель процесора та обсяг оперативної пам'яті
  • Автоматичне визначення Windows 10 / Windows 11 для правильного вибору версії утиліт
  • Кнопки "Обрати все" та "Зняти виділення" для зручного керування списком
  • Запускається лише з правами адміністратора — програма автоматично запитує підвищення привілеїв
  • Налаштування Notepad++, WinRAR та інших програм постачаються у вигляді готових пакетів
