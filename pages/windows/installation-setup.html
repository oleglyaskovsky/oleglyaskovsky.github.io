Установка та налаштування Windows

Покрокова інструкція з чистої установки Windows 10/11 з автоматичним налаштуванням системи
Про що ця стаття

Це мій особистий алгоритм чистої установки Windows: від завантаження образу до готової системи, яка після першого ввімкнення вже має потрібний обліковий запис, вимкнену зайву телеметрію, видалені непотрібні вбудовані застосунки, свої шпалери на робочому столі та ярлик програми для швидкого встановлення решти софту. Все це працює за рахунок файлу автовідповідей, який Windows читає ще під час установки, тож майже всі кроки майстра установки проходять автоматично, без ручних кліків.

1
Завантажте образ Windows

Завантажте останню доступну версію офіційного ISO-образу Windows 10 або Windows 11. Зручніше за все брати посилання через сервіс massgrave.dev — там завжди актуальні прямі посилання на образи від Microsoft, без стороннього втручання.

Образи Windows 10 Образи Windows 11
2
Згенеруйте файл автовідповідей (autounattend.xml)

Щоб установка пройшла максимально «чисто» і без зайвих запитань майстра, а система одразу отримала потрібні налаштування, я користуюсь онлайн-генератором на schneegans.de. Він створює файл autounattend.xml, який Windows автоматично зчитує під час установки. У своєму конфігу я налаштовую:

  • Автоматичне створення локального облікового запису адміністратора без зайвих екранів входу в Microsoft-акаунт
  • Випадкове ім'я комп'ютера за власним шаблоном
  • Вимкнення телеметрії, SmartScreen, UAC, автоматичних перезавантажень та інших системних «дратівливих» речей
  • Видалення переважної більшості вбудованих застосунків (Xbox, Cortana, OneDrive, Teams, Copilot, Paint 3D тощо)
  • Темну тему оформлення, класичне контекстне меню та зручний вигляд Провідника й панелі завдань
Відкрити генератор із базовим конфігом

Налаштувавши всі потрібні пункти, натисніть кнопку завантаження на сторінці генератора — ви отримаєте готовий файл autounattend.xml. Його потрібно покласти в корінь ISO-образу — це зроблю на кроці 4.

3
Підготуйте додаткові файли: шпалери та Software Installer

Перед тим як об'єднати все в один ISO-образ, потрібно підготувати таку структуру папок і файлів у себе на комп'ютері — назвемо цю папку Windows_ISO_Update. Вона повторює структуру системних папок Windows, тому все, що в ній лежить, після установки автоматично опиниться у потрібному місці на диску:

autounattend.xml ← файл з кроку 2, кладемо поруч із sources/ sources/ └── $OEM$/ ├── $$/ │ └── Web/ │ └── Wallpaper/ │ └── MyWallpaper.jpg ← покладіть сюди зображення з такою самою назвою — і воно стане шпалерами одразу після установки └── $1/ └── Users/ └── Public/ └── Desktop/ └── Software Installer.exe ← завантажте актуальну версію за посиланням нижче і покладіть сюди

$$ у Windows OEM-розгортанні означає системну папку Windows, а $1 — корінь системного диску. Тобто файл із $$\Web\Wallpaper після установки опиниться в C:\Windows\Web\Wallpaper, а програма з $1\Users\Public\Desktop — просто на робочому столі. Software Installer — це мій власний застосунок для швидкого встановлення базового набору програм одразу після переустановки системи.

Завантажити Software Installer
4
Об'єднайте все в один ISO-образ (AnyBurn)

Тепер потрібно «вшити» вміст папки Windows_ISO_Update у завантажений на кроці 1 образ Windows. Для цього я користуюсь безкоштовною портативною програмою AnyBurn: у ній є режим редагування ISO-образу, який дозволяє додати autounattend.xml у корінь образу та об'єднати папку sources з тією, що вже є всередині ISO. Після цього зберігаю результат як новий ISO-файл.

Завантажити AnyBurn Free Portable
5
Створіть мультизавантажувальну флешку через Ventoy

Ventoy дозволяє один раз підготувати флешку, а потім просто перекидати на неї будь-яку кількість ISO-образів як звичайні файли — без повторного форматування чи перезапису щоразу, коли з'являється новий образ. Завантажте Ventoy, встановіть його на флешку (це відформатує накопичувач), а вже потім скопіюйте на неї потрібні ISO-файли.

Завантажити Ventoy
Порада. Копіювання нового ISO-образу на готову Ventoy-флешку — це просто перетягування файлу. Саму флешку переробляти повторно не потрібно.
6
Скопіюйте готовий образ на флешку

Залишається перекинути готовий ISO-файл на флешку з Ventoy (крок 5) — просто перетягуванням, як звичайний файл, без будь-якого повторного форматування. Під час завантаження з флешки Ventoy покаже меню вибору образу: обираєте потрібний ISO — і встановлення Windows проходить вже майже без вашої участі, за налаштованим на кроці 2 сценарієм.
